Sistema de Monitoramento e Gestão de Energia

O SMGE é um sistema inteligente que permite monitorar o consumo e os custos com energia dos equipamentos e principais setores de uma unidade comercial ou industrial.

O Sistema apresenta o perfil do gasto com energia, permitindo ao usuário identificar as otimizações cabíveis no processo com base na eficiência energética dos equipamentos da instalação.

Através das telas, o usuário visualizará os custos e consumo permitindo que o usuário tenha uma maior previsão do custo com energia e/ou ajuste o processo de produção visando uma eventual economia na conta de energia.

Vale ressaltar que o sistema é estruturado de forma customizada, atendendo as necessidades de cada cliente, relevando o porte da empresa, bem como o perfil de consumo energético.
